PokeDynamic v1.0 - Windows 灵动岛 (耿鬼版)
============================================

感谢您下载 PokeDynamic！

【功能特点】
• 耿鬼主题灵动岛 - 可爱的耿鬼陪你工作学习
• 系统监控 - 实时显示 CPU/GPU 温度和负载
• 音乐控制 - 自动检测播放音乐并显示歌词
• 通知镜像 - 显示 Windows 通知
• AI 对话 - 耿鬼陪你聊天解闷
• 潜伏模式 - 闲置时显示耿鬼眼睛

【系统要求】
• Windows 10 1903+ 或 Windows 11
• .NET 6.0 Runtime 或更高版本
• 4GB RAM (推荐 8GB)

【快捷键】
• Alt + L - 切换潜伏模式
• Alt + E - 切换眼睛颜色
• Alt + Q - 显示系统状态
• Alt + P - 番茄钟模式
• Alt + T - 切换主题
• 双击 - 展开/收起聊天

【使用说明】
1. 安装完成后运行 PokeDynamic
2. 首次运行会引导您完成配置
3. 享受您的耿鬼灵动岛体验！

【支持】
如有问题，请访问: https://github.com/pokedynamic
