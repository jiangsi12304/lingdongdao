## 📖 简介

PokeDynamic 是一款 Windows 桌面应用，将 macOS 的灵动岛体验带到 Windows 平台，并融入了可爱的耿鬼主题。它可以在你的桌面顶部显示一个动态的灵动岛，提供系统监控、媒体控制、通知镜像等功能。

## ✨ 功能

- 🎵 **音乐控制** - 自动检测正在播放的音乐并显示歌词
- 📊 **系统监控** - 实时显示 CPU/GPU 温度和负载
- 🔔 **通知镜像** - 显示 Windows 通知并获取发送者头像
- 🤖 **AI 对话** - 耿鬼陪你聊天解闷 (支持豆包 AI)
- 🌤️ **天气显示** - 显示当前位置天气信息
- ⏱️ **番茄钟** - 25分钟专注工作计时器
- 👁️ **潜伏模式** - 60秒无操作后进入省电模式 (仅显示眼睛)
- 🎨 **主题切换** - 多种颜色主题可选
- 📁 **文件拖放** - 拖放文件到灵动岛快速操作

## 💻 系统要求

### 最低配置
- Windows 10 1903+ 或 Windows 11
- Intel Core i3 / AMD Ryzen 3
- 4GB RAM
- 100MB 可用空间
- .NET 6.0 Runtime

### 推荐配置
- Windows 11 22H2+
- Intel Core i5 / AMD Ryzen 5
- 8GB RAM
- 200MB 可用空间
- .NET 8.0 Runtime

## 📥 下载

访问 [Releases](https://github.com/pokedynamic/pokedynamic/releases) 页面下载最新版本。

## 📦 安装

1. 下载 `PokeDynamic-Setup-v1.0.exe`
2. 双击运行安装程序
3. 选择安装路径
4. 选择是否创建桌面快捷方式和开机自启
5. 完成安装

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt + L` | 切换潜伏模式 |
| `Alt + E` | 切换眼睛颜色 |
| `Alt + Q` | 显示系统状态 |
| `Alt + P` | 番茄钟模式 |
| `Alt + T` | 切换主题 |
| 双击 | 展开/收起聊天 |

## 🛠️ 技术栈

- **前端**: React + TypeScript + Tailwind CSS
- **后端**: C# / .NET 9.0
- **AI**: 豆包 API

## 📝 开发

```bash
# 克隆仓库
git clone https://github.com/pokedynamic/pokedynamic.git

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建
npm run build
```
